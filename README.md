Ementa:  
- Fases da Resolução de um Problema via Computador  
- Estruturas Básicas  
- Comandos de Seleção  
- Comandos de Repetição  
- Procedimentos e Funções  
- Arquivos  
- Estruturas de dados em Python  
- Introdução a bibliotecas científicas
- Projeto final
